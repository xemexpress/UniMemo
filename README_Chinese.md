# UniMemo 發掘順便

### TL;DR

以商品實付金額的10%作為酬金，順便代買&送遞。

### <u>Memo紙</u>

- 承諾按金
- 代買清單
- 交收時地

| 關鍵配套     | 作用                                  |
| ---------- | ------------------------------------- |
| 電子支付平台 | <u>建立信任</u>。提供安心墊支的合作前提。   |
| 合作商戶     | <u>保障權益</u>。承諾為未查收的商品退款。  |
| 雙向評準     | <u>重視意見</u>。低評的互動會被紀錄並跟進。|
| 隱匿通訊    | <u>尊重私隱</u>。確保個人資料時刻保密。    |
| 路線演算     | <u>連接順便</u>。搜尋最相關的Memo。      |
