# 關於安全

用戶的安全是我們最重視的一環。

1. **身份證及人臉認證**

   在提供電郵和手機號碼進行註冊後，你需要上載你的永久性居民身份證圖片（清晰可見、完整四角）和頭像照片（清晰、完整臉部、不戴太陽眼鏡及帽）作進一步認證。

2. **一鍵警報**

   若交收期間有需要，你可使用一鍵警報，把你的位置和Memo互動詳情向執法部門呈報並求助。

3. **Memo互動分享**

   你可與親友分享你的即時位置和Memo互動狀態。

4. **雙向評準**

   低評的Memo互動將會被紀錄並跟進，相關用戶將可能被移除。
